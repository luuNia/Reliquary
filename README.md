# Reliquary

Reliquary is a lightweight Minecraft server administration plugin for Paper/Purpur (1.21.x).

It allows admins to inspect container activity and see who placed or removed items from chests and similar inventories.

## Command
/co inspect

![Inspect in action](main/ddd.png)

## Built With
- Java
- Paper / Purpur API

## Installation
1. Download the latest release
2. Put the `.jar` file into your `plugins/` folder
3. Restart the server
